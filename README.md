# DSK-Shop 

Bienvenue sur le dépôt de **DSK-Shop**, une plateforme e-commerce moderne inspirée du design minimaliste.

## 🛠 Technologies utilisées
* **Frontend :** HTML5, Tailwind CSS, JavaScript (ES6 Modules)
* **Icônes :** Lucide Icons
* **Design :** Inspiré par Skena.co

## 📦 Fonctionnalités
- [x] Panier dynamique
- [x] Filtres par catégories (Gaming, Audio, Tech)
- [x] Design Responsive (Mobile-first)
- [ ] Système de paiement Mobile Money (En cours)A simple HTML/JS/CSS starter template
